📌 Pattern Generator and Number Analyzer (Python)
📖 Overview

This is a simple Python console-based project that allows users to:

	• Generate star (*) patterns
	• Analyze a range of numbers (even, odd, sum, total count)

It is designed as a beginner-friendly project to understand:

	• Loops
	• Conditional statements
	• User input handling
	• Lists in Python
🚀 Features
  🔹 Pattern Generator
	• Right Triangle Pattern
	• Inverted Triangle Pattern
  🔹 Number Analyzer
	• Finds even numbers
	• Finds odd numbers
	• Calculates total numbers
	• Calculates sum of numbers
  🔹 Menu Driven Program
	• Easy-to-use interactive menu
	• Continuous execution until user exits
🛠️ Technologies Used
	• Python (Core Concepts)
	• No external libraries required

💡 How It Works
	1️⃣ Main Menu

	User selects:
	• 1 → Pattern Generator
	• 2 → Number Analyzer
	• 3 → Exit
	2️⃣ Pattern Logic
	✔ Right Triangle
	*
	**
	***
	****
	✔ Inverted Triangle
	****
	***
	**
	*
	3️⃣ Number Analysis Example

	Input:

	• Start = 1
	• End = 10

	Output:
	• Even Numbers: [2, 4, 6, 8, 10]
	• Odd Numbers: [1, 3, 5, 7, 9]
	• Total Numbers: 10
	• Sum: 55
